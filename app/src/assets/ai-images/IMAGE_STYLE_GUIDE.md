# Polymind AI Image Style Guide

## Brand Colors (Use in Prompts)
```
Background:    #0A0A0B (deep void black)
Surface:       #111113 (dark charcoal)
Primary:       #F59E0B (amber gold)
Golden:        #F7C948 (warm gold)
Accent Purple: #8B5CF6 (lavender)
Accent Blue:   #3B82F6 (electric blue)
Success:       #22C55E (sage green)
Coral:         #FF6B6B (warm coral)
```

## Unified Art Style
**Style Name:** "Cosmic Neural Minimalism"

**Core Elements:**
- Dark, nearly black backgrounds (#0A0A0B)
- Glowing amber/gold as the hero color
- Subtle particle systems and light trails
- Abstract neural/constellation patterns
- Flowing, ethereal energy streams
- Sacred geometry undertones
- Depth through subtle gradients

**Mood:** Mysterious, expansive, empowering, futuristic yet timeless

**Avoid:**
- Photorealistic imagery
- Cartoonish or playful styles
- Overly busy compositions
- Bright/white backgrounds
- Generic stock photo aesthetics

## Prompt Template
```
[Subject description], cosmic neural minimalism style,
dark void background (#0A0A0B), glowing amber gold (#F59E0B) accents,
ethereal light particles, abstract and minimal,
subtle purple (#8B5CF6) undertones, cinematic lighting,
8k, ultra detailed, trending on artstation
```
